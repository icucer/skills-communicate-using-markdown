### Example in Python
```python
def greet(name):
    print(f"Hello, {name}!")

greet("World")
```

### Example in Java
```java
public class helloWorld {
    public static void greet(String name) {
        System.out.println("Hello, " + name + "!");
    }

    public static void main(String[] args) {
        greet("World");
    }
}
```

### Example in JavaScript
```javascript
function greet(name) {
    console.log(`Hello, ${name}!`);
}

greet("World");
```
